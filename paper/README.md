## Data provenance
Every number in the tables was transcribed from the committed `study.ipynb`
outputs and cross-checked programmatically (persistence sanity values,
per-model NRMSE/RPS both conditions, compute budget, simulated + real weather
deltas, Nemenyi ranks, published-value comparison). The real +weather results
(Table 5) use the validated 7-channel Open-Meteo ERA5 acquisition described in
§3.5; all 10 source locations passed the temperature-alignment gate
(r in [0.958, 0.995], offset < 1.1 C).
